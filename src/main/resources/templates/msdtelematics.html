Msd Telematics

This Is basically a webapp in which we have to Maintain the record of various accounts of the Excise Department of bihar and UP government. The Main Code Part of the project is done in spring core language. Some new functionalities were added by using Springboot Module . We Have also Used Erlang in this project to deal with Real Time socket Data . We have also used Kafka as a messaging service .

In this project, the main technologies used by us are basically Springboot,rest API's , MYSQL , Erlang,Kafka, Redis and Spring Core. There were two servers in the deployment of the project . The first one is development server and the other one is production server . These servers were maintained by devops team but all the work regarding production was done by backend developers only.

There was also a provision of sending alerts. Then only the main role of kafka , erlang and redis comes into play. ERlang is used maintain the sockets and these sockets are fitted as devices into the vehicles . All the data from these sockets is maintained by erlang only . The storage of this data is been done in redis database. All the dynamic data ais been stored in redis database and all the static data is been stored in MYSQL dtabase.

All the task of sending and creating the alerts is been maintained by kafka engine . All the code of this kafka engine is been done in spring core as a moderator.
